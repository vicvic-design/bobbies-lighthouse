# Bobbies Lighthouse 1.0.1

## Compatibility

- Minecraft: 1.21.10
- Fabric Loader: 0.17.x
- Fabric API: 0.138.3+1.21.10
- Bobby: required
- Sodium: optional compatibility path included

## Notes

- Developed with AI assistance; review before use and use at your own risk.
- Latest known jar for the lighthouse rewrite.
- Client-only Fabric mod.
- Uses lodestone anchors to request Bobby cached chunks around discovered lighthouse points.
- Bobby integration uses reflection against Bobby internals and may require compatibility updates as Bobby changes.
